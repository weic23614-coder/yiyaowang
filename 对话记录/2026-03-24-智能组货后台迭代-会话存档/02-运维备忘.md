# 运维备忘

## 重启 uvicorn

1. 运行服务的终端里 **Ctrl+C** 停止。  
2. 再执行（**`export` 行不要带中文**，注释另起一行或删掉）：

```bash
cd /Users/weipeng/zhinengzuhuo-backup
source .venv/bin/activate
export APP_DB_PATH="$PWD/app_runtime.db"
uvicorn app.main:app --host 127.0.0.1 --port 8098
```

## 端口被占用（Errno 48）

```bash
lsof -i :8098
kill <PID>
```

再启动 uvicorn。若页面已能打开，说明旧进程仍在跑；要加载新代码必须先 kill 再启。
