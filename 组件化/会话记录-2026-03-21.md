### 7. Git push 被拒与 `git pull` 报错 `#`

- **用户**（终端日志）：`~/yiyaowang` 与 `~/药网组件化代运营` 分别 `init` 后 `push` 被拒；`git pull origin main --rebase   # 若远程已有提交` 出现 `fatal: couldn't find remote ref #`。  
- **助手**：**原因 1**：远程已有提交，需 `pull --allow-unrelated-histories` 或 **`push --force-with-lease`**（以本地为准时）。**原因 2**：行尾 **`#` 注释**若字符非 ASCII `#`，可能被当成 ref。建议命令与注释**分行**写。
